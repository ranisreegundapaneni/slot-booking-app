# Slot Booking App

This project is a simple slot booking application built using React.js for the frontend and Node.js with Express for the backend.

The application allows a user to see the available time slots, book a slot by entering their name, and view the slots that are already booked.

Features
- Display available time slots
- Allow user to book a slot
- Show the slots that are already booked

Main Files
- App.js – Handles the main UI and displays the slots
- api.js – Used for calling backend APIs
- server.js – Backend server that manages slot data and booking

How to Run the Project

Start Backend Server
node server.js

Start Frontend
npm start
